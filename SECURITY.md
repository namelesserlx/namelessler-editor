# Security Policy

## Supported Versions

Security fixes are provided for the latest development state of this repository
and the latest published package version.

| Version            | Supported |
| ------------------ | --------- |
| `main`             | Yes       |
| latest npm release | Yes       |
| older releases     | No        |

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Instead, report vulnerabilities privately by emailing:

- `namelesslx@foxmail.com`

When possible, include:

- a clear description of the issue
- steps to reproduce
- impact assessment
- affected versions or commit range
- any suggested mitigation

You can expect an initial response within a reasonable time after the report is received.
